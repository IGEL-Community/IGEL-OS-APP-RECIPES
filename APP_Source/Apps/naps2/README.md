# [NAPS2 - Not Another PDF Scanner](https://www.naps2.com/)

NAPS2 is free and open source scanning software for Linux.

Easily scan with devices from Canon, Brother, HP, Epson, Fujitsu, and more. Then save to PDF, TIFF, JPEG, or PNG with a single click.

- Download Linux Installer (x64) deb file and rename as `naps2-linux-x64.deb`
