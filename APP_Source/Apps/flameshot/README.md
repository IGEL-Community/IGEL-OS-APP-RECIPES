# [Flameshot](https://flameshot.org/)

- **NOTE:** Version 14.0.0+ uses GLIBC version 2.38. IGEL OS 12 is currently 2.36 so you need to use version 13.3.0. To find current version of GLIBC on IGEL OS:

```bash linenums="1"
ldd --version
```

- Obtain Flameshot for Linux AppImage and zip it up - [Linux AppImage Download Link for 13.3.0](https://github.com/flameshot-org/flameshot/releases/tag/v13.3.0)

- Rename file to `Flameshot.AppImage` and zip it into `Flameshot.AppImage.zip`
